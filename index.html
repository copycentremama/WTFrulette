    ПРАВИЛА ИГРЫ

    Manual & Versions

    ОСНОВЫ

    • Игроков: 2–8. Каждый делает 3 хода за раунд.
    • Белый — 25 очков, Чёрный — 50, Красный — 100.
    • Очки сбрасываются после каждого раунда.
    • При отказе наказание придумывают другие игроки.
    • В конце раунда игрок с наименьшим счётом вылетает.
    • При ничьей — тайбрейк: задание на вылет.

    КОЛЕСО

    • 10 секторов: Белый / Чёрный / Красный.
    • Вероятности растут каждый раунд: больше чёрных и красных.
    • В последнем раунде — только Чёрные и Красные (0% белых).
    • Адаптация: меньше игроков — выше шанс красных.

    ВЕРСИИ ИГРЫ

    🎲 БАЗОВАЯ Лёгкий флирт, весёлый дискомфорт, безумные шутки. 18+
    🔥 18+ ТРЕШ Максимальный риск, стыдные задания, потеря репутации.
    🔐 ЧАСТНАЯ КОЛЛЕКЦИЯ Только по коду. 21+. Самые откровенные и смелые задания.

    ДОСТУП 21+

    • Требуется подтверждение возраста (21+).
    • LUX-версия доступна только по коду.
    • Код выдаётся лично организатором вечеринки.
    • Подтверждение сохраняется — не нужно вводить каждый раз.
